# busdriver

A simple card game.

## Run locally

Prerequisits

node and npm installed

Preparation

1. Clone repository
2. Enter client folder
3. run "npm install"
4. Enter server folder
5. run npm install

Start local server

6. Enter client folder
7. run "npm run serve"
8. Enter server folder
9. run "npm start"

Now you can access the game at http://localhost:8080/
